## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app locally

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app in docker

```bash
# start in detached mode
$ docker-compose up -d

# start redis image
$ docker-compose up redis

# start api image
$ docker-compose up nest-api
```
